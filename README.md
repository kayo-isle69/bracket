# BRACKET — Tournament Manager

> Free. Offline. No account. No ads.

A double elimination tournament bracket manager built for real people who run real tournaments at gaming cafes, LAN parties, and local competitions.

**→ Live: [kayo-isle69.github.io/bracket](https://kayo-isle69.github.io/bracket)**

---

## Features

- ✅ Works completely offline — no internet required after first load
- 🏆 Full double elimination (Winners + Losers bracket + Grand Final reset)
- 🎲 4 / 8 / 16 / 32 players, with optional random seeding
- 💾 Tournaments saved locally in your browser — resume anytime
- 👆 One-tap to advance winners, losers auto-route to correct bracket slots
- 🚫 Zero ads. Zero fees. Zero sign-in. Forever.
